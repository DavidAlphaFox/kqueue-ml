Module Kqueue.Kevent

Kevent.t is used to represent an input/output event that can be registered with the kernel. A kevent consists of:

  • ident: an identifier for an event.
  • filter: filter is used to determine if an event has met the pre-condition for it to be made available for retrieval by the user.
  • flags: flags are used to specify actions that need to be performed on an event.
  • fflags: filter specific flags.
  • data: filter specific data.
  • udata: User-defined data that gets passed through unchanged.
